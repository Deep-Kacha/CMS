## ğŸš€ Getting Started

### Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download) installed on your machine
- [Visual Studio](https://visualstudio.microsoft.com/) (recommended) or VS Code with C# extension

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Deep-Kacha/CMS.git
   cd CMS
   ```

2. **Open in Visual Studio**
   - Open the `.sln` file inside either the `CMS/` or `CIE2_CMS_Project/` folder.

3. **Build the project**
   - Use `Build > Build Solution` in Visual Studio, or run:
   ```bash
   dotnet build
   ```

4. **Run the application**
   ```bash
   dotnet run
   ```
